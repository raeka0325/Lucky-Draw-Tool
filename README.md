# 🎉 尾牙抽獎系統

一個簡單易用的網頁抽獎工具，適合公司尾牙、活動抽獎使用。

## ✨ 功能特色

### 📥 三種匯入方式

1. **上傳 Excel 檔案**（推薦）
   - 支援 .xlsx 和 .xls 格式
   - 自動讀取第一列作為參與者姓名
   - 最穩定可靠的方式

2. **手動貼上名單**
   - 從 Excel 複製貼上
   - 一行一個姓名
   - 最快速的方式

3. **貼上雲端連結**
   - 支援 Google Drive 分享連結
   - 支援 Dropbox 分享連結
   - 支援 Google Sheets 公開連結
   - 只需上傳到雲端，複製分享連結即可
   - 注意：本地使用可能受 CORS 限制

### 🎊 抽獎功能

- ✅ 炫酷的滾動抽獎動畫
- ✅ 自訂獎項名稱（頭獎、貳獎、參獎等）
- ✅ 設定每個獎項的中獎人數
- ✅ 已中獎者不會重複中獎
- ✅ 查看所有得獎名單
- ✅ 顯示剩餘參與者人數
- ✅ **NEW** 可選擇性匯入獎項清單（Excel 或手動貼上）
- ✅ **NEW** 自動按獎項順序抽獎，顯示當前獎項進度

## 📖 使用說明

### 1. 準備參與者名單

**方法一：上傳 Excel 檔案（推薦）**
1. 準備 Excel 檔案，第一列（A欄）為參與者姓名
2. 點擊「選擇檔案」上傳或直接拖曳檔案

**方法二：手動貼上**
1. 從 Excel 複製姓名欄位
2. 貼到文字框（一行一個姓名）

**方法三：使用雲端連結**

#### Google Sheets
1. 在 Google Sheets 第一列（A欄）輸入參與者姓名
2. 點擊「共用」→ 設定為「知道連結的人都能查看」
3. 複製分享連結
4. 貼到抽獎系統的「貼上雲端連結」欄位

#### Google Drive
1. 上傳 Excel 檔案到 Google Drive
2. 右鍵點擊檔案 → 「取得連結」
3. 設定為「知道連結的人都能查看」
4. 複製連結並貼到系統

#### Dropbox
1. 上傳 Excel 檔案到 Dropbox
2. 點擊「共用」→ 「建立連結」
3. 複製分享連結並貼到系統

### 2. 匯入獎項清單（可選）

匯入參與者名單後，可以選擇性地匯入獎項清單，系統會自動按順序進行抽獎。

**啟用/停用功能：**
- 獎項清單區域右上角有開關，預設為「開啟」
- 關閉開關將隱藏獎項匯入功能，並清除已匯入的獎項
- 如果不需要獎項功能，可直接關閉

**方法 A：上傳 Excel 檔案**
1. 準備 Excel 檔案，包含兩欄：
   - 第一欄：獎項名稱（例如：特獎、頭獎、貳獎）
   - 第二欄：中獎人數（例如：1、2、5）
2. 點擊「下載範例」查看正確格式
3. 上傳檔案

**方法 B：手動貼上**
1. 按照格式輸入：`獎項名稱:人數`
2. 一行一個獎項，例如：
   ```
   特獎:1
   頭獎:2
   貳獎:5
   參獎:10
   ```
3. 點擊「匯入獎項」

### 3. 開始抽獎

**有匯入獎項清單：**
1. 點擊「確認開始抽獎」
2. 系統會自動顯示第一個獎項資訊
3. 點擊「開始抽獎」開始抽獎
4. 系統顯示滾動動畫後公布中獎者
5. 當前獎項抽完後，點擊「繼續抽下一個獎項」
6. 重複步驟 3-5，直到所有獎項抽完

**沒有匯入獎項清單：**
1. 點擊「確認開始抽獎」
2. 點擊「開始抽獎」
3. 系統顯示滾動動畫後公布中獎者
4. 可以重複抽獎，直到所有參與者抽完

### 4. 查看結果

- 已中獎者會即時顯示在「已抽中名單」區域
- 如果有匯入獎項清單，每位中獎者會標示所屬獎項
- 已中獎者不會重複中獎
- 剩餘人數會即時更新

## 📝 Excel 格式範例

### 參與者名單範例

```
姓名（必填） | Email（選填）
張三         | zhang@example.com
李四         | li@example.com
王五         | wang@example.com
趙六         | zhao@example.com
錢七         |
```

### 獎項清單範例

```
獎項名稱 | 中獎人數
特獎     | 1
頭獎     | 2
貳獎     | 5
參獎     | 10
```

點擊「📥 下載範例檔」可以直接下載包含正確格式的 Excel 範例檔案。

## 🔧 技術說明

- 純前端實現，無需伺服器
- 使用 SheetJS (xlsx.js) 解析 Excel
- 支援跨域讀取雲端檔案（需公開分享）
- 響應式設計，支援手機和電腦

## ⚠️ 注意事項

1. **雲端連結必須設為公開分享**
   - Google Drive/Sheets：「知道連結的人都能查看」
   - Dropbox：使用分享連結

2. **Excel 格式要求**
   - 第一列必須是姓名
   - 避免空白行
   - 建議去除表頭（或第一行會被當作參與者）

3. **跨域限制**
   - 某些雲端服務可能有跨域限制（CORS）
   - 本地使用時，建議優先使用「方法一：上傳檔案」
   - 如使用雲端連結遇到問題，請下載檔案後改用上傳方式

## 🎯 使用技巧

- **建議先匯入獎項清單**：這樣系統會自動顯示當前獎項，更有儀式感
- 可以在同一個活動中抽多個不同獎項
- 建議先測試小範圍名單，確認功能正常
- 獎項清單可以隨時清除
- 已中獎者會顯示所屬獎項標籤（如有匯入獎項清單）
- 系統會自動追蹤獎項進度，顯示「第 X / Y 個獎項」
- 如果不匯入獎項清單，也可以直接開始抽獎，適合簡單場合使用
